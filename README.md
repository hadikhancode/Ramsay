# Ramsay
A web app that shows recipes based on selected food preferences using Gemini AI and AllRecipes

---

### Instructions to run:
- Install [Google Cloud CLI](https://docs.cloud.google.com/sdk/docs/install-sdk#latest-version)
- run `pip install -r requirements.txt`
- run `gcloud auth application-default login`
- rename .env.example to .env and edit the values
- run `py backend/main.py`
